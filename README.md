# PowerBI-Portfolio-Project

## Overview

This repository contains a collection of interactive Power BI dashboards developed for business intelligence and data analytics. The project demonstrates data visualization, KPI tracking, sales performance analysis, profitability insights, geographical analysis, and financial reporting using Power BI Desktop.

## Project Objectives

- Transform raw business data into meaningful insights.
- Build interactive dashboards for decision-making.
- Analyze sales, profit, discounts, and product performance.
- Track Year-over-Year (YOY) growth trends.
- Visualize regional and geographical business performance.

---

## Dashboards Included

### 1. YOY Sales Growth Dashboard
- Total Sales Analysis
- Year-over-Year Growth Percentage
- Trend Analysis (2014–2017)
- Comparative Sales Performance

### 2. Waterfall Analysis Dashboard
- Profit contribution by category
- Technology, Furniture, and Office Supplies performance
- Annual profit movement tracking

### 3. Location Wise Sales Dashboard
- City-wise Sales Analysis
- Geographic performance comparison
- Top-performing locations identification

### 4. Sales, Profit & COGS Dashboard
- Quarterly Sales Analysis
- Profit Tracking
- Cost of Goods Sold (COGS) Monitoring
- Year → Quarter → Month Drill-down functionality

### 5. Regional Profit Dashboard
- Profit Distribution by Region
- West, East, South, and Central region analysis
- Percentage contribution visualization

### 6. Finance Dashboard
Key KPIs:
- Total Sales
- Total Profit
- Units Sold
- Discount Analysis

Features:
- Time-series sales analysis
- Segment-wise performance
- Profitability tracking
- Discount distribution insights

### 7. Product Performance Dashboard
- Product-wise Sales Analysis
- Product-wise Profit Analysis
- Units Sold Comparison
- Discount Distribution by Product
- Product Contribution Analysis

### 8. Sub-Category Analysis Dashboard
- Sales by Sub-Category
- Profit by Sub-Category
- Quantity Sold
- Discount Analysis
- Top and Bottom Performing Categories

---

## Key Insights

### Sales Performance
- Strong year-over-year sales growth observed.
- Consistent increase in overall revenue.

### Regional Analysis
- West region contributes the highest profit share.
- East region follows as the second-highest contributor.

### Product Analysis
- Paseo emerges as the highest-selling product.
- Product profitability varies significantly across categories.

### Financial Insights
- Enterprise and Government segments generate major revenue.
- Discount management directly impacts profitability.
